# Adaptive Data Science Copilot with LLM

An AI-powered adaptive data science system that automatically analyzes datasets, detects machine learning tasks, benchmarks multiple ML models, and generates intelligent explanations using Large Language Models (LLMs).

---

# Features

- Automatic Dataset Analysis
- Intelligent Task Detection
- Adaptive Model Benchmarking
- AutoML Workflow
- Multiple ML Model Comparison
- AI-Generated Explanations
- Explainable AI
- Interactive Web Interface

---

# Supported Tasks

## Classification
- Fraud Detection
- Spam Detection
- Customer Churn Prediction

## Regression
- House Price Prediction
- Sales Prediction
- Revenue Forecasting

## Clustering
- Customer Segmentation
- Market Analysis

## Time-Series
- Sales Forecasting
- Stock Prediction
